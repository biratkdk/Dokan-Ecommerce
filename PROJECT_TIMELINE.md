# Project Timeline

## Original Final-Year Project (2022)

This project began in 2022 as a final-year undergraduate ecommerce system developed with Python and Django.

### Original 2022 scope

- product catalog and product detail pages
- user registration and login
- cart and checkout workflow
- order placement and order history
- admin-side product and order management
- relational database integration

## Post-2022 Enhancements

After the original academic project was completed, the system was further improved and maintained as an extended portfolio application.

### Later enhancements

- payment gateway integration
- email verification and notification flows
- support ticket and customer conversation workflow
- analytics dashboard and operational monitoring
- recommendation and customer insight modules
- API endpoints and health checks
- deployment configuration and production hardening

## Repository Note

The current repository is not intended to represent a frozen 2022 submission. It represents the original project together with later maintenance and feature extensions.
